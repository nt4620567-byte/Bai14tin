Câu hỏi 1 trang 75

Đề bài: Nếu muốn tất cả các đoạn văn bản của trang web có màu xanh (blue) thì cần thiết lập CSS thế nào?

Lời giải:

p {
    color: blue;
}

Câu hỏi 2 trang 74

Đề bài: Nếu muốn thiết lập CSS để áp dụng cho nhiều trang web thì làm cách nào?

Lời giải: Dùng CSS ngoài (external CSS) hoặc CSS nội tuyến (inline CSS).

Luyện tập 1 trang 75

Khẳng định: “CSS chính là HTML” đúng hay sai?

Lời giải: Sai. CSS và HTML là hai ngôn ngữ khác nhau.

Luyện tập 2 trang 75

Khẳng định: “Chỉ cần thay đổi tệp CSS là làm thay đổi định dạng nhiều trang web”.

Lời giải: Đúng.

Vận dụng 1 trang 75

Đề bài: So sánh Style Sheet trong Word và CSS.

Lời giải:

  • Cả hai có cấu trúc quy tắc định dạng.
  • Có tính tái sử dụng.
  • Tạo khuôn mẫu định dạng.
  • Dùng để kiểm soát phong cách văn bản.

Vận dụng 2 trang 75

Yêu cầu: Tạo trang web “Lịch sử CSS”.

Lịch sử CSS

Ý tưởng CSS do kỹ sư Hakon Wium Lie (Na Uy) đưa ra năm 1994 khi làm việc tại CERN.

Mục đích của CSS là tách biệt nội dung HTML và phần định dạng.

Các phiên bản đầu tiên

• CSS1 ra đời năm 1996.

• CSS2 hoàn thiện năm 1998.

• CSS2.1 ra đời năm 2011, bản nâng cấp năm 2016.

Các phiên bản tiếp theo

• Từ CSS3 trở đi được phát triển theo từng module.

• Các chuẩn CSS4 và CSS5 đang được phát triển.

• Tổ chức phát triển: W3C – www.w3.org.
